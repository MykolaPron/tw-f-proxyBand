## Run Locally

Clone the project

```bash
  git clone https://github.com/MykolaPron/tw-f-proxyBand
```

Go to the project directory

```bash
  cd tw-f-proxyBand
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  npm run build
  npm run start
```
Start the dev server

```bash
  npm run dev
```
